# LLM Usage Report

## Tools used

- Kimi Code CLI (Large Language Model coding assistant)
- GitHub Copilot / inline suggestions were not used.

## How they were used

- Assisted with code generation for FastAPI endpoints, Pydantic schemas, store/service logic, frontend React components and TypeScript API clients.
- Helped write test cases and verify test failures before implementing fixes.
- Generated project documentation, report templates and the CHANGELOG entry.
- Suggested the stacked-PR branching strategy and issue numbering conventions.

## Human oversight

All generated code was reviewed, tested and adjusted by the development team. The team decided on the MVP v1 scope, acceptance criteria, estimates and design trade-offs. AI output was used as a starting point, not as a final deliverable.
