# LLM and AI Usage Report

The following AI tools were used during Assignment 2:

## Whisper

The Whisper `large-v3` model was used to automatically transcribe the customer meeting recording from Russian speech into text. The resulting transcript was manually reviewed and corrected by the team.

## ChatGPT

ChatGPT was used for:

* translating the customer meeting transcript from Russian into English;
* correcting transcription errors and improving readability;
* structuring and editing the Week 2 report files;
* drafting the Figma prototype prompts and refining the web-interface design description.

## Figma Make / AI-assisted design

The interactive Figma prototype was created with Figma's AI-assisted design features. The team provided detailed prompts describing the required screens, workflows, and visual style; Figma generated the initial interactive prototype and an optional React + TypeScript + Tailwind CSS code export. The prototype is published publicly; the generated code is kept locally and is not part of the repository.

## Code Agent

A Code Agent (Kimi Code CLI) was used to partially assist with:

* implementing the FastAPI microservice wrapper around the existing solver;
* writing Docker and Docker Compose configuration for the API;
* generating and reviewing Markdown documentation (user stories, interface docs, MVP v0 report, analysis, and this LLM report);
* checking links and placeholders across the Week 2 deliverables.

All AI-generated content was reviewed and edited by the team before submission.
